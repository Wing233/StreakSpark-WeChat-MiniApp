# 图标文件说明

微信小程序仅支持 .png、.jpg、.jpeg 格式的图标文件。
请手动创建以下 PNG 格式的图标文件：

1. list.png - 任务列表图标（未选中状态）
2. list_selected.png - 任务列表图标（选中状态）
3. add.png - 添加任务图标（未选中状态）
4. add_selected.png - 添加任务图标（选中状态）
5. about.png - 关于页面图标（未选中状态）
6. about_selected.png - 关于页面图标（选中状态）

建议图标尺寸：40x40px 或 60x60px

颜色建议：
- 未选中状态：#8a8a8a
- 选中状态：#3498db
